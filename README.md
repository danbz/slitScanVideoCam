# SlitscanClock

code in C++ using openFrameworks.cc for timeLapse 24hr slitscan clock
// for example in oF core 
#Controls

* 1: scan horizontal
* 2: scan vertical
* 3: ribbon horizontal
* 4: ribbon vertical
* 5: slitscan clock 
* c: toggle debug and preview
* f: toggle fullscreen
* r: toggle to draw anystyle scan as radial
* t: toggle drawing hour and minute thumbnails
